# Kill or Build

A polished, client-only MVP web app that helps users decide whether an AI project should be **Kill**, **Redesign**, or **Build**.

## Stack

- Vite
- React
- TypeScript
- plain CSS
- Vitest

## Run locally

```bash
npm install
npm run dev
```

## Test

```bash
npm test
```

## Build

```bash
npm run build
```

No backend, auth, persistence, payments, or AI API.
